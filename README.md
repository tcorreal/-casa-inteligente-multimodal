# Casa Inteligente Multimodal

Proyecto de interfaces multimodales con Streamlit y Teachable Machine.

## Tecnologías

- Python + Streamlit
- TensorFlow (modelo de Teachable Machine)
- Control multimodal:
  - Controles gráficos (botones, sliders, toggles)
  - Comandos de texto
  - Control por gestos con cámara
